# A5 DuckDB Extension

Hierarchical pentagonal indexing — equal-area, millimeter-accurate geospatial cells across 31 resolution levels.

> Extension: `a5` (FROM community). Maintainer: rustyconover (Query.farm). MIT License.

## Setup

```sql
INSTALL a5 FROM community;
LOAD a5;
```

## Key Properties

- **Equal-area cells** — unlike H3 hexagons, A5 pentagons have exactly equal area at each resolution (OGC DGGS compliant)
- **31 resolution levels** (0–30) — from continental (42M km²) down to sub-millimeter (0.03 mm²)
- **12 base cells** at resolution 0 covering the globe
- **Pentagonal geometry** — no distortion artifacts from hexagonal singularities

## Coordinate Conversion

```sql
-- Point → A5 cell (note: latitude first, longitude second)
SELECT a5_lonlat_to_cell(lat, lon, 15) AS cell_id;

-- A5 cell → center point [longitude, latitude]
SELECT a5_cell_to_lonlat(cell_id);  -- returns DOUBLE[2]

-- Get resolution from cell ID
SELECT a5_get_resolution(cell_id);
```

## Cell Hierarchy

```sql
-- Parent at coarser resolution
SELECT a5_cell_to_parent(cell_id, 10);

-- All children at finer resolution (returns UBIGINT[])
SELECT a5_cell_to_children(cell_id, 20);
```

## Cell Boundary (for visualization)

```sql
-- Boundary vertices as DOUBLE[2][] (array of [lon, lat] pairs)
SELECT a5_cell_to_boundary(cell_id);

-- Control ring closure and segment count
SELECT a5_cell_to_boundary(cell_id, true, 10);  -- closed=true, segments=10
-- segments: auto-calculated if ≤0; increase for smoother curves
```

## Cell Metrics

```sql
-- Area in square meters for a resolution level
SELECT a5_cell_area(15);  -- ~32 hectares at res 15

-- Total cells at a resolution
SELECT a5_get_num_cells(15);
```

## Compaction

```sql
-- Compact: replace complete sibling groups with parent cell
SELECT a5_compact(cell_list);

-- Uncompact: expand cells back to target resolution
SELECT a5_uncompact(compacted_list, 20);
```

## Enumeration

```sql
-- All 12 base cells (resolution 0)
SELECT a5_get_res0_cells();
```

## Resolution Guide

| Resolution | Cell Area (approx) | Use Case |
|------------|-------------------|----------|
| 0–5 | 42M km² – 33k km² | Continental / Country |
| 6–10 | 8k km² – 130 km² | Regional / State |
| 11–15 | 32 km² – 32 hectares | City / District |
| 16–20 | 8 hectares – 124 m² | Neighborhood / Building |
| 21–25 | 31 m² – 0.5 m² | Room / Vehicle |
| 26–30 | 8 cm² – 0.03 mm² | Precision measurements |

## A5 vs H3 Comparison

| Property | A5 | H3 |
|----------|----|----|
| Cell shape | Pentagon | Hexagon (+ 12 pentagons) |
| Equal area | Exact | Approximate (~3.5% variation) |
| Resolutions | 31 (0–30) | 16 (0–15) |
| Finest cell | ~0.03 mm² | ~0.9 m² |
| Base cells | 12 | 122 |
| Standard | OGC DGGS | Uber proprietary |
| Grid ops (k-ring, path) | Not available | Available |
| Directed edges | Not available | Available |

## Common Patterns

```sql
-- Index points to A5 cells and aggregate
SELECT a5_lonlat_to_cell(lat, lon, 12) AS cell,
       count() AS cnt,
       avg(value) AS avg_val
FROM points
GROUP BY ALL;

-- Spatial join via A5 cell matching
SELECT a.*, b.*
FROM table_a a
JOIN table_b b
  ON a5_lonlat_to_cell(a.lat, a.lon, 15) = a5_lonlat_to_cell(b.lat, b.lon, 15);

-- Convert boundary to WKT for use with spatial extension
-- (requires manual polygon construction from boundary vertices)
SELECT cell_id, a5_cell_to_boundary(cell_id) AS vertices
FROM cells;

-- Multi-resolution aggregation using compaction
WITH cells AS (
    SELECT a5_lonlat_to_cell(lat, lon, 20) AS cell_id
    FROM events
)
SELECT unnest(a5_compact(list(cell_id))) AS compacted_cell
FROM cells;
```

## Complete Function Reference

| Function | Signature | Description |
|----------|-----------|-------------|
| `a5_lonlat_to_cell` | `UBIGINT (lat DOUBLE, lon DOUBLE, res INT)` | Coordinates → cell ID |
| `a5_cell_to_lonlat` | `DOUBLE[2] (cell UBIGINT)` | Cell → center [lon, lat] |
| `a5_cell_area` | `DOUBLE (res INT)` | Cell area in m² |
| `a5_cell_to_boundary` | `DOUBLE[2][] (cell UBIGINT [, closed BOOL [, segments INT]])` | Boundary vertices |
| `a5_cell_to_parent` | `UBIGINT (cell UBIGINT, target_res INT)` | Parent cell |
| `a5_cell_to_children` | `UBIGINT[] (cell UBIGINT, target_res INT)` | All children |
| `a5_get_resolution` | `INT (cell UBIGINT)` | Extract resolution |
| `a5_get_num_cells` | `UBIGINT (res INT)` | Total cells at resolution |
| `a5_get_res0_cells` | `UBIGINT[] ()` | All 12 base cells |
| `a5_compact` | `UBIGINT[] (cells UBIGINT[])` | Merge siblings → parent |
| `a5_uncompact` | `UBIGINT[] (cells UBIGINT[], target_res INT)` | Expand to resolution |
