# A5 Pentagonal & S2/Geography Extension Functions

Alternative spatial indexing and spherical geometry systems for DuckDB.

---

## S2 / Geography Extension Functions (Spherical Geometry)

> Extension: `geography` (FROM community). Source: [paleolimbot/duckdb-geography](https://github.com/paleolimbot/duckdb-geography).
> All functions operate on the **sphere** with geodesic edges. Prefix: `s2_` (not `st_`).

### Custom Types

| Type | Storage | Description |
|------|---------|-------------|
| `GEOGRAPHY` | BLOB (WKB) | Spherical geometry with geodesic edges (same as BigQuery GEOGRAPHY) |
| `S2_CELL` | UINT64 | S2 hierarchical cell ID (31 levels, ~2cm finest) |
| `S2_CELL_CENTER` | UINT64 | Point as cell center (8-byte compact encoding) |
| `S2_CELL_UNION` | UINT64[] | Normalized cell list for polygon approximation |
